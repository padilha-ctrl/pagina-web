Requisitos do Sistema

Clique abaixo para acessar rapidamente:

  • Requisitos Funcionais (RF)
  • Requisitos Não Funcionais (RNF)

Requisitos Funcionais (RF)

RF01 - Funcional: O sistema deve permitir que o aluno acesse o acervo de livros.
Prioridade: Alta

RF02 - Funcional: O sistema necessita que se tenha login dos usuários (aluno, professor, bibliotecário).
Prioridade: Alta

RF03 - Funcional: O sistema deve ter funcionalidades como reservar e devolver livros.
Prioridade: Alta

RF04 - Funcional: O sistema deve avisar quando o aluno deve devolver o livro.
Prioridade: Média

RF05 - Funcional: O sistema deve registrar a devolução do livro e o estado do mesmo.
Prioridade: Baixa

RF06 - Funcional: O sistema deve ter ferramentas para adicionar livros novos.
Prioridade: Alta

RF07 - Funcional: O sistema deve ter páginas referentes a cada gênero de conhecimento (Livro físico/online, Podcast, Audiobooks).
Prioridade: Média

RF08 - Funcional: O sistema deve disponibilizar uma página referente às reservas online.
Prioridade: Baixa

Requisitos Não Funcionais (RNF)

RNF01 - Não Funcional: O sistema precisa ser acessível em dispositivos móveis.
Prioridade: Alta

RNF02 - Não Funcional: O sistema precisa ser acessível em sites Web.
Prioridade: Alta

RNF03 - Não Funcional: O sistema precisa ter segurança para todos os usuários.
Prioridade: Alta

RNF04 - Não Funcional: O sistema deve permanecer no “ar” toda hora.
Prioridade: Baixa
